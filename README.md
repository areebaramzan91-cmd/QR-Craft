QR Craft

QR Craft is a Python-based QR Code Generation system designed to create high-quality and customizable QR codes efficiently.
The project demonstrates practical implementation of Python libraries for data encoding and image generation while maintaining a clean and scalable project structure.

Overview

This application converts user-provided data such as URLs, text, or identifiers into scannable QR codes and exports them as image files. The project is lightweight, fast, and structured for easy enhancement and future scalability.

Core Features

* Dynamic QR Code Generation
* High-Resolution Image Export
* Lightweight and Optimized Workflow
* Clean and Maintainable Python Code
* Easy Integration into Larger Applications

Technology Stack

* Python
* qrcode
* Pillow (PIL)

Installation

Clone the repository:

bash id="c1"
git clone https://github.com/your-username/QR-Craft.git

Navigate to the project directory:

bash id="c2"
cd QR-Craft

Install dependencies:

bash id="c3"
pip install qrcode[pil]

Execution

Run the application using:

bash id="c4"
python qr_generator.py

The generated QR code image will automatically be stored in the project directory.
